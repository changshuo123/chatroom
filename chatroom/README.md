# chatroom

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Run your unit tests
```
npm run test:unit
```

### Lints and fixes files
```
npm run lint
```

### 项目目录
```
res 表情文件
src 源码目录
    --api 请求接口
    --assets 静态资源文件
    --components 通用组件
    --config 全局配置
    --plugins 插件配置
    --router 路由信息
    --sdk 云信sdk
    --static 静态资源文件
    --store vuex仓库
    --utils 工具文件
    --views 视图
    --App.vue 跟组件
    --main.js 入口js文件
vue.config.js webpack配置
```
